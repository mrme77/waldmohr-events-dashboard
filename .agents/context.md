# Agent Context

This project is a local English events dashboard for an American family moving to Waldmohr.

Key decisions:
- Static app first.
- Source data comes from public Waldmohr Aktuell WordPress REST API.
- Every event needs a source URL and freshness metadata.
- Dates inferred from post text must be marked with confidence.
